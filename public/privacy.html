Visão Geral da Privacidade

Conformidade LGPD

Nosso sistema está em total conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018).

Padrões Internacionais

Seguimos as melhores práticas internacionais de proteção de dados, incluindo GDPR.

Certificações

Possuímos certificações ISO 27001 e SOC 2 Type II para segurança da informação.

Treinamento

Nossa equipe recebe treinamento regular sobre proteção de dados e privacidade.

Princípios da LGPD que Seguimos
Finalidade

Coletamos dados apenas para finalidades específicas, explícitas e legítimas relacionadas ao funcionamento do restaurante.

Adequação

O tratamento de dados é compatível com as finalidades informadas ao titular dos dados.

Necessidade

Limitamos o tratamento ao mínimo necessário para a realização de suas finalidades.

Livre Acesso

Garantimos consulta facilitada e gratuita sobre a forma e duração do tratamento.

Transparência

Fornecemos informações claras, precisas e facilmente acessíveis sobre o tratamento.

Coleta e Uso de Dados

Dados Pessoais de Clientes
  • Nome completo: Para identificação do pedido
  • Telefone: Para contato sobre o pedido
  • Email: Para envio de confirmações e promoções (com consentimento)
  • Endereço: Para entrega de pedidos
Dados de Pedidos
  • Histórico de pedidos: Para melhorar a experiência do cliente
  • Preferências alimentares: Para recomendações personalizadas
  • Método de pagamento: Para processamento de transações
  • Avaliações: Para melhoria contínua do serviço
Dados de Uso
  • Logs de acesso: Para segurança e auditoria
  • Dados de navegação: Para melhorar a interface
  • Cookies técnicos: Para funcionamento do sistema
  • Estatísticas de uso: Para análise de performance
Dados de Funcionários
  • Dados cadastrais: Para gestão de recursos humanos
  • Dados de acesso: Para controle de permissões
  • Registros de atividade: Para auditoria interna
  • Dados de treinamento: Para capacitação em LGPD
Base Legal para Tratamento
Execução de Contrato

Dados necessários para o cumprimento de contrato ou procedimentos preliminares relacionados a contrato.

Consentimento

Dados coletados mediante consentimento livre, informado e inequívoco do titular.

Interesse Legítimo

Para atender aos interesses legítimos do controlador ou de terceiro, respeitando os direitos do titular.

Cumprimento Legal

Para o cumprimento de obrigação legal ou regulatória pelo controlador.

Segurança e Proteção de Dados

Criptografia

Todos os dados são criptografados em trânsito (TLS 1.3) e em repouso (AES-256).

Infraestrutura

Servidores hospedados em data centers certificados ISO 27001 no Brasil.

Controle de Acesso

Acesso restrito por função, autenticação multifator e logs de auditoria.

Backup e Recuperação

Backups automáticos diários com retenção segura e testes de recuperação.

Medidas de Segurança Implementadas
Pseudonimização e Anonimização

Dados pessoais são pseudonimizados sempre que possível e anonimizados para análises estatísticas.

Minimização de Dados

Coletamos apenas os dados estritamente necessários para as finalidades declaradas.

Retenção Limitada

Dados são mantidos apenas pelo tempo necessário, com exclusão automática após o período de retenção.

Monitoramento Contínuo

Sistemas de monitoramento 24/7 para detectar e responder a incidentes de segurança.

Testes de Segurança

Testes de penetração regulares e avaliações de vulnerabilidade por empresas especializadas.
